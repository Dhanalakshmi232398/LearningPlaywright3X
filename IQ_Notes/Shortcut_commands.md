| Action | Windows | Mac |
|---|---|---|
| Command Palette | `Ctrl + Shift + P` | `Cmd + Shift + P` |

| Settings | `Ctrl + ,` | `Cmd + ,` |
| Keyboard Shortcuts | `Ctrl + K Ctrl + S` | `Cmd + K Cmd + S` |
